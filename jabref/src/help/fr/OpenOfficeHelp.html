Exporter vers OpenOffice.org

JabRef peut exporter votre base de données dans le format classeur d'OpenOffice.org. Le classeur exporté contiendra une feuille de calcul listant les entrées selon les lignes et les différents champs selon les colonnes. Le classeur est compatible avec les fonctions bibliographiques d'OpenOffice.org, ce qui signifie que vous pouvez paramétrer le classeur exporté comme votre base de données bibliographique en utilisant Outils -> Sources de données...
